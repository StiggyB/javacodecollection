Detailed Description

Core Controller

SE2 (+ SY and PL) Project SoSe 2011

Authors: Rico Flaegel, Tell Mueller-Pettenpohl, Torsten Krane, Jan Quenzel

Capsulates many functions for the direct in- and output from and to the Festo Transfersystem and with Interrupts using Pulse Messages. Starts the threads. Implements Singleton-Pattern.

Upcoming: Other parts can get their necessary ChannelID's.
