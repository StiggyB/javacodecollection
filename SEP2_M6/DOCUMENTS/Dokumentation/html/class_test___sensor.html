Test_Sensor

SE2 (+ SY and PL) Project SoSe 2011

Authors: Rico Flaegel, Tell Mueller-Pettenpohl, Torsten Krane, Jan Quenzel

Tests the sensor system. React to the messages from the interrupt controller. Additional testing of expected values and included reaction from the system.

Tests:

Tests the light sensor B(0) Tests the light sensor B(1) Tests the height-measure B(2) Tests the light sensor B(3) Tests the inductive sensor B(4) Tests the magnetic sensor B(5) Tests the light sensor B(6) Tests the light sensor B(7)

Tests the start button C(4) Tests the stop button C(5) Tests the reset button C(6) Tests the emergency button C(7)
